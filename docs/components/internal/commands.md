# Internal Commands Reference

This section documents the built-in CLI commands implemented in `internal/cmd`.

-   **[Generate](commands/generate.md)**: Scaffolding for projects, commands, and flags.
-   **[Regenerate](commands/regenerate.md)**: Synchronizing manifest and code.
-   **[Remove](commands/remove.md)**: Deleting components safely.

!!! note "Developer Usage"
    These commands are run via `go run main.go [command]` during development.
